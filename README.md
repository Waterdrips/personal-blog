## Usage

### Locally

`bundle exec jekyll serve`

### Production

`JEKYLL_ENV=production bundle exec jekyll build`

## Install


```shell
$ bundle install
$ yarn install

$ yarn run tailwind init _includes/tailwind.config.js
```
